# Welcome!

Welcome to the Fintech and Cryptocurrencies Practical Exam. This wiki provides you with information and links to concepts required for this exam as well as the objectives and requirements.

The exam is based on Fungible Tokens (ERC20 standard) and Decentralized Exchanges (DEXs) using Uniswap V3. You will apply what you've learned from the tutorials and assignments to complete this practical assessment.

> **NOTE:** This is an individual exam and plagiarism will be penalized. Any new concepts will have links provided for you to read through before attempting the exam. Please read the wiki for the detailed exam requirements.

## Requirements

- The submission is your private repository
- Exam duration: **3 hours**
- You must complete all TODO items in the codebase
- All tests must pass when running `yarn test:local`
- Complete UI demonstration with screenshots

> **IMPORTANT:** The exam focuses on implementing SWAP transactions for DEXs using Uniswap V3, which was not covered in Assignment 2.

If you have any issues during the exam, please contact the instructor: takunda.chirema@uct.ac.za
